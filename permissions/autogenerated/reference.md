## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-start-listening`
- `allow-stop-listening`
- `allow-set-window-labels`
- `allow-set-event-types`
- `allow-is-listening`
- `allow-key`
- `allow-button`
- `allow-move-mouse`
- `allow-scroll`
- `allow-text`
